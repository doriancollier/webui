# DorkOS

A web-based chat interface and REST/SSE API for [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview), built on the Claude Agent SDK.

DorkOS gives you a browser UI for Claude Code sessions â€” with real-time streaming, tool approval flows, slash command discovery, and multi-client sync. Use it standalone via npm, embed it in Obsidian, or build your own client against the API.
